Non-Component view.dtd elements

The following table lists the Gui4j elements that do not represent GUI components.

Name
Description
Arg
Instantiation for parameters with Param
ClassAlias
A ClassAlias is used to declare Java classes to gui4j.
Include
Include other xml definition
Param
Parameters to be substituted when including this file
Ref
Placeholder for a referenced component.
Style
Define a new style
Style_XXX
Style definition for tag XXX. Only allowed inside Style
StyleBegin
Begin of a style block
StyleEnd
End of a style block
View
Top level xml element (see extra page)
