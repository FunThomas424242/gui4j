Parameters in xml files

All referenced components in a xml description must exist except those specified in Param elements:

<Param id="button"/>

In this example a parameter named button is introduced. Other descriptions in the same file might refer to this name. When including this file with Include a value for this parameter must be provided using the Arg element:

...
<Include ...>
  <Arg param="button" id="myCloseButton"/>
</Include>
...
<button guiId="myCloseButton" .../>

 This concept can be used to provide GUI templates (see the Tips & Tricks section). See also Include for the usage about Arg and Param.
