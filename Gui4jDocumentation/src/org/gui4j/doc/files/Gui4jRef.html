Ref
Description
The Ref element acts as a reference to a top-level component with a certain guiId. You can use this element and reference a top-level component defined elsewhere in your XML-file instead of having to define the other top-level component in place. This is not relevant for the functionality of your GUI but may be helpful to keep your XML files readable.
Sub-Elements
NONE
Required Attributes
id
Gui4j-ID
The guiId of the referenced component.
