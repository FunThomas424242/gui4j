Basic Structure

The root elment of each gui4j description is the element View. A usual xml file looks as following:

<?xml version="1.0" encoding="iso-8859-1"?>
<!DOCTYPE View SYSTEM "view.dtd">
  <View>
    ... some definitions
  </View>

 The file view.dtd is the used DTD (available here). See View for more information about the root element.
